Background

Adding color to element's background

You can add the background color to any element doing any of the following

Lets say we were going to use two colors, blue, and orange-700

Declare the full color as an attribute
<div color-blue></div>
<div color-orange-700></div>
Declare the color name within the color attribute
<div color="blue"></div>
<div color="orange-700"></div>
Declare the color name using the CSS class
<div class="color-blue"></div>
<div class="color-orange-700"></div>
