Buttons

Adding color to button backgrounds

You can add the background color to any <button> or <a> doing any of the following

Lets say we were going to use two colors, green, and red-400

Declare the full color as an attribute
<button btn-green></button>
<button btn-red-400></button>
Declare the color name within the color attribute
<button btn-color="green"></button>
<button btn-color="red-400"></button>
Declare the color name using the CSS class
<button class="btn-green"></button>
<button class="btn-red-400"></button>
