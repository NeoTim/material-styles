Typography

Adding color to your text.

You can add the color to any element doing any of the following

Lets say we were going to use two colors, purple, and pink-300

Declare the full color as an attribute
<div text-purple></div>
<div text-pink-300></div>
Declare the color name within the color attribute
<div text-color="purple"></div>
<div text-color="pink-300"></div>
Declare the color name using the CSS class
<div class="text-purple"></div>
<div class="text-pink-300"></div>
