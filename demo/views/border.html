Border Colors

Adding color to element's border

You can add the border color to any element doing any of the following

Lets say we were going to use two colors, purple, and lime-A400

Declare the full color as an attribute
<div color-purple></div>
<div color-lime-A400></div>
Declare the color name within the color attribute
<div color="purple"></div>
<div color="lime-A400"></div>
Declare the color name using the CSS class
<div class="color-purple"></div>
<div class="color-lime-A400"></div>
{{color.name}}
